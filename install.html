Chrome · Edge · Brave — Unpacked Install

  1. Download the ZIP

    Click "Download ZIP" above to save vortex07-2.6.0.zip to your computer.

  2. Extract the folder

    Right-click the ZIP and choose "Extract All". Put the resulting folder somewhere permanent — moving it later will break the extension.

  3. Open the Extensions page

    In Chrome or Edge, navigate to chrome://extensions. In Brave, go to brave://extensions.

  4. Enable Developer Mode

    Toggle the Developer mode switch in the top-right corner of the extensions page. The toolbar will show new buttons.

  5. Load Unpacked

    Click Load unpacked, then select the extracted vortex07-2.6.0 folder (the one containing manifest.json).

  6. Visit playvortex.io

    Navigate to playvortex.io and do a hard refresh (Ctrl+Shift+R / Cmd+Shift+R). The retro theme should appear immediately.

Firefox — Temporary Install

  1. Download & extract the ZIP

    Same as steps 1–2 above. Extract to a permanent location.

  2. Open about:debugging

    In Firefox, type about:debugging in the address bar and press Enter.

  3. This Firefox → Load Temporary Add-on

    Click This Firefox in the left sidebar, then click Load Temporary Add-on… and select the manifest.json file inside the extracted folder.

  4. Visit playvortex.io

    Navigate to playvortex.io and hard refresh. Note: Firefox temporary add-ons are removed when the browser is closed — you'll need to re-load after each browser restart.

Notes & Troubleshooting

  • The extension only activates on playvortex.io — it has no effect on other sites.
  • If the theme doesn't appear, try a full hard refresh: Ctrl+Shift+R (Windows/Linux) or Cmd+Shift+R (Mac).
  • Make sure you selected the folder containing manifest.json (not the ZIP itself, and not a parent folder).
  • Moving the extension folder after loading will break it — you'll need to "Load unpacked" again.
  • Reputation votes require the Vortex07 API at vortex07.vercel.app — they won't work offline.
  • To update, replace the folder contents with the new ZIP and click the reload icon on chrome://extensions.
