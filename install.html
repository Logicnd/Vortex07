Download Vortex07

Chrome · Edge · Brave

  1. Extract the ZIP somewhere permanent.
  2. Open chrome://extensions.
  3. Turn on Developer mode.
  4. Load unpacked → pick folder with manifest.json.
  5. Visit playvortex.io, hard refresh.

Firefox

  1. Same extract as above.
  2. about:debugging → This Firefox.
  3. Load Temporary Add-on → pick manifest.json.
  4. Reload after every browser restart.

Wrong folder? Pick the one with manifest.json, not the ZIP. Hard refresh: Ctrl+Shift+R.
