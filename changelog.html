Version History

v2.6.0 June 2026 Current
  • content.js refactor: 10,383 lines split into 2,124-line core + 8 focused modules — hover-ui, rep-ui, game-rating-ui, profile-ui, guestbook-ui, layout-ui, search-ui, friends-ui
  • styles.css refactor: 6,773 lines replaced by a 9-line @import root + 9 CSS partials — tokens, dark, jvortex, reset, layout, components, friends, profile, forum
  • Carousel arrow fix: Friends-grid was being wrapped inside the .section-header DOM node — now extracted before carousel build, arrows display correctly
  • IP daily rate limit: 15 reputation votes per IP per day — prevents alt-account farming
  • Cloudflare Turnstile CAPTCHA: Wired into reputation vote submission with graceful degradation when not configured
  • isVoteMethod bug fix: Variable was undefined — vote POSTs were silently returning 405 Method Not Allowed; votes now register correctly
  • Admin reset-all endpoint: POST /api/rep-admin with { "action": "reset-all" } — wipes all reputation data (admin-key protected)
  • Icon asset resize: All three icons were 1024×1024 (~3 MB total); corrected to 16×16, 48×48, and 128×128 px
v2.5.0 June 2026
  • Account-bound rep votes: actorUserId now attached to every vote — reputation is tied to accounts, not just IPs
  • HMAC-signed requests: Service worker signs all API calls; server rejects unsigned or tampered requests
  • Friends A–Z sort & pagination: 10 friends per page, alphabetically ordered, with left/right arrow navigation
  • Social friend search: Search friends by display name directly from the social panel
  • Dev audit panel: Spike detection dashboard for reputation admins — flags unusual voting bursts
  • Dark mode overhaul: Complete rework of the dark/night theme — zero pure-white elements, consistent purple depth
v2.3.0 June 2026
  • Community Forum tab: Built-in discussion board powered by the Vortex07 API — browse and post threads without leaving playvortex.io
  • Forum threads, replies, and author data stored server-side via Vercel serverless functions
  • Forum API available at GET /api/forum (thread list) and POST /api/forum (new thread)
v2.2.0 June 2026
  • Game Like / Dislike voting: Thumbs-up / thumbs-down buttons injected on game detail pages — votes tracked globally via the API
  • Vote tallies displayed alongside the game title in XP-style UI
v2.1.0 June 2026
  • Compare Users: Side-by-side stat comparison panel for any two players
  • Roulette button: Jump to a random friend's profile with one click
  • Game page polish — section headers, XP chrome, retro typography applied to all game views
  • Hover card improvements — faster load, smoother positioning
v2.0.0 June 2026 — Initial Release
  • Visual unification: Consistent 2007 Roblox / Windows XP Luna aesthetic across all playvortex.io pages
  • Route parity: Theme applied to home, profile, social, game, search, and catalog pages
  • Section header bars — purple gradient with bold white uppercase labels
  • XP chrome panels — bevelled borders, subtle drop shadows
  • Global reputation system — vote on players, leaderboard, API-backed storage
  • Popup with per-feature toggles and settings
  • Service worker background script for API communication
  • Manifest V3 — compatible with Chrome, Edge, Brave, and Firefox
