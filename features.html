🎨 Visual / Theme

🟣
2007 Roblox Section Headers Purple gradient bars with bold white uppercase labels — identical to the original 2007 Roblox site.
🪟
Windows XP Chrome Panels have bevelled borders (lighter top/left, darker bottom/right) and a subtle drop shadow, exactly like XP Luna windows.
🔤
Retro System Fonts Body text in Tahoma/Verdana; headers in Arial Black — the authentic early-2000s web stack, no Google Fonts.
🌑
Retro Night / Dark Mode Deep purple tones replace every white surface. Zero pure-white elements — stays true to the XP night aesthetic.
🖱
XP-Style Buttons Outset bevel border on idle, inset on press. Gold download buttons match the era's call-to-action style.
🖼
CSS Token System All colours defined as CSS custom properties — easy to fork and retheme with a single file change.

👥 Friends & Social

📋
A–Z Friends Carousel Your friends list is sorted alphabetically and displayed 10 at a time, with left/right arrow navigation.
🔍
Friend Search Search your friends by name in real-time from the social panel.
🎲
Roulette Button Jump to a random friend's profile page with a single click.
⚖
Compare Users Side-by-side stat comparison between any two players on playvortex.io.
👁
Profile Hover Cards Hover over any player name to see a quick-view card with stats and reputation.

⭐ Reputation System

👍
Vote on Any Player Upvote or downvote any player's reputation. Votes are tied to your account ID and stored server-side.
🏆
Global Leaderboard See the top-rated players in the Vortex07 community, sorted by net reputation score.
🛡
Anti-Alt-Farm Protection IP-based daily rate limit (15 votes/IP/day) plus Cloudflare Turnstile CAPTCHA to prevent vote manipulation.
🔐
HMAC-Signed Requests Reputation votes are signed by the extension's service worker — server rejects unauthenticated submissions.
📊
Audit Panel Developer audit panel with spike detection to identify unusual voting patterns.

💬 Forum

📌
Built-In Forum Tab A community discussion board rendered directly inside the extension — no separate website required.
✍
Post & Reply Create threads and reply from the Forum tab inside the extension. The API is shared with this site.
🌐
Live on This Site The forum thread list is readable here at forum.html — same data, no login needed.

⚙ Settings & Customisation

🔧
Per-Feature Toggles Enable or disable individual features (reputation display, friends carousel, forum tab, etc.) from the popup.
💾
Local Storage Only All settings are saved in chrome.storage.local — they sync across your tabs but never leave your device.
🎛
Theme Toggle Switch between light (XP Luna) and dark (Retro Night) from the extension popup.

🛠 Developer / Admin

🔑
Admin Reset Endpoint POST /api/rep-admin with { "action": "reset-all" } wipes all reputation data. Admin-key protected.
📡
Health Check GET /api/health returns API status and uptime metrics.
🏗
Open Source Full source on GitHub. The API is serverless (Vercel functions) and the extension is plain JS — no build step required.
