Privacy Policy

Effective date: June 2026  ·  Applies to: Vortex07 browser extension and this website.

What We Collect

Vortex07 does not collect, store, or transmit personal data. No names, emails, IP addresses (beyond standard server logs), or personal information of any kind are gathered by this extension or website.

Extension Settings

All extension settings — theme preference, enabled features, and UI options — are stored exclusively in your browser's local storage via chrome.storage.local. This data never leaves your device and is not accessible to us.

Reputation System

The optional reputation voting feature sends only the following to the Vortex07 API:

  • The numeric user ID of the player being voted on (a public identifier from playvortex.io)
  • The numeric user ID of the voter (your public playvortex.io ID)
  • The vote value (+1 or −1)

No display names, emails, or personal information are transmitted. The API uses IP addresses transiently for rate-limiting (max 15 votes/IP/day) but does not log or store them persistently.

Forum

Forum threads and posts are stored by the Vortex07 API and are publicly readable. Posts contain only the text you write and your playvortex.io user ID. Do not include personal information in forum posts.

Analytics & Tracking

There are no analytics, tracking pixels, cookies, or third-party scripts on this website or in the browser extension. No data is shared with advertising networks or third parties.

Third-Party Services

The Vortex07 API is hosted on Vercel, which may collect standard server access logs (IP address, timestamp, URL) as part of their infrastructure. These logs are subject to Vercel's privacy policy. Reputation vote submissions optionally pass through Cloudflare Turnstile for bot protection — Turnstile is a privacy-preserving CAPTCHA that does not use persistent cookies or fingerprinting.

Open Source

The full source code for the extension and API is available on GitHub. You can audit exactly what data is sent and when.

Contact

Questions about this policy? Open an issue on the GitHub repository.
