What's new in v2.6.0

  • content.js split into 8 focused modules — was 10,383 lines, core is now 2,124 (hover-ui, rep-ui, game-rating-ui, profile-ui, guestbook-ui, layout-ui, search-ui, friends-ui)
  • styles.css split into 9 CSS partials — tokens, dark, jvortex, reset, layout, components, friends, profile, forum
  • Carousel arrow DOM bug fixed — friends-grid was being wrapped inside the section header node
  • Cloudflare Turnstile CAPTCHA added to reputation votes with graceful degradation
  • IP daily rate limiting — 15 rep votes per IP per day (anti-alt-farm)
  • isVoteMethod undefined bug fixed — vote POSTs were silently returning 405, votes now work correctly
  • Admin reset-all endpoint: POST /api/rep-admin with { action: "reset-all" }
  • Icon assets resized from 1024×1024 (~3 MB) to correct 16 / 48 / 128 px dimensions

Features

🎨
2007 Roblox Aesthetic

Section headers, XP chrome borders, retro purple palette, and pixel-perfect fonts.

👥
Friends Carousel

A–Z sorted friends list, 10 per page with arrow navigation and live search by name.

⭐
Global Reputation

Vote on any player, see a community leaderboard, and track your own score.

💬
Community Forum

Built-in discussion tab powered by the Vortex07 API — no external site needed.

🌑
Retro Night Theme

Zero white in dark mode. Deep purple tones that stay true to the XP aesthetic.

🔒
Privacy-First

No tracking, no account required. Settings stored locally in your browser only.
