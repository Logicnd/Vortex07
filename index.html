Vortex07

A small Chrome extension for playvortex.io. Old-school 2007-style look, plus rep, forum, and a few QoL tweaks.

Download the latest build

Download Vortex07 How to install

ZIP for Chrome, Edge, Brave, or Firefox. Unpacked install.

Quick install

  1. Download & extract the ZIP.
  2. chrome://extensions → Developer mode → Load unpacked.
  3. Open playvortex.io and hard refresh.
